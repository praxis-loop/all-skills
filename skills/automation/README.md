# automation

`automation` 用于可重复执行的自动化流程，重点是让 agent 按固定步骤操作工具、设备、本地应用、定时任务或外部服务。

适合放入本分类的 skill：

- 手机、浏览器、桌面应用、命令行工具的自动化操作。
- 定时或半自动任务，例如打卡、巡检、同步、通知。
- 有明确前置条件、执行步骤和成功/失败判断的 workflow。

不适合放入本分类的 skill：

- 主要产出文字报告的流程，放到 `reports`。
- 主要处理代码仓库和 CI 的流程，放到 `coding`。
- 主要面向基础设施运维的流程，放到 `operations`。
