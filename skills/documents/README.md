# documents

`documents` 用于正式文档、表格、幻灯片、PDF、合同、表单和知识库内容生产。

适合放入本分类的 skill：

- Word、Markdown、Excel、PowerPoint、PDF 的创建、编辑和检查。
- 合同、制度、SOP、知识库、模板化文档产出。
- 文档格式、结构、目录、术语和最终交付检查。

不适合放入本分类的 skill：

- 轻量日报和周报，放到 `reports`。
- 数据清洗和指标计算，放到 `data`。
- 视觉素材生成和图片处理，放到 `media`。

当前暂无 skill。新增时应明确目标文件格式、模板来源和验收标准。
