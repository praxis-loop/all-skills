# ecommerce

`ecommerce` 用于电商运营、商品内容、平台规则和销售转化相关流程。

适合放入本分类的 skill：

- Amazon、Shopify、独立站、跨境平台 Listing 优化。
- 标题、五点、描述、A+ 页面、关键词和 SEO 工作流。
- 产品图、卖点提炼、竞品分析和平台合规检查。

不适合放入本分类的 skill：

- 纯视觉资产生产流程，放到 `media`。
- 纯数据分析或报表查询流程，放到 `data`。
- 通用市场研究流程，放到 `research`。

当前暂无 skill。新增时请先参考 `docs/SKILL_GUIDE.md`。
